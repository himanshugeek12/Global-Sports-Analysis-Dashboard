# Global-Sports-Analysis (Interactive Dashboard creation using MS Excel)
## Project Objective
The objective of the Global Sports Analysis Dashboard is to analyze sports performance, player distribution, and country-wise targets across the globe. This dashboard helps stakeholders understand which sports and regions are performing better and where growth opportunities exist.

## Dataset used 
- <a href="https://github.com/himanshugeek12/Global-Sports-Analysis-Dashboard/blob/master/Global_Sports_Analytics_Dataset.xlsx">Dataset</a>

## ‚ùì Key Questions (KPIs)
- What is the **total score across all sports**?
- Which **major sports contribute the highest scores**?
- Which **minor sports are performing strongly**?
- How are **targets distributed across countries**?
- Which **regions have the highest registered players**?
- Comparison between **National, International, and Both** categories using filters.

-Dashboard Interaction <a href="https://github.com/himanshugeek12/Global-Sports-Analysis Dashboard/blob/master/Global%20Sports%20Dashboard.png">View Dashboard</a>


## üîÑ Process
- Verified data for missing values and inconsistencies.
- Cleaned and formatted data for analysis.
- Created calculated fields for total scores and player counts.
- Built interactive visuals for:

  - Total score by minor sports
  - Country-wise target distribution

## üîç Project Insights
- **Rugby and Cricket** are the top-performing major sports.
- **Squash and Kho-Kho** lead among minor sports.
- **Australia and Russia** have the highest target scores.
- **Central Asia and South America** show the highest player registrations.
- Several regions show growth potential due to lower participation.
